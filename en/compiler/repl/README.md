# Interactive REPL Engine (`src/repl/`)

This directory documents the Read-Eval-Print Loop (REPL) engine, which provides an interactive execution shell for the XCX compiler and VM runtime.

## Directory Index

- **[repl.md](repl.md):** Details the interactive loop architecture, state persistence (`SymbolTable`, `Interner`, and `VM` instances), multi-line input editing with console shortcuts, and diagnostic REPL commands.
