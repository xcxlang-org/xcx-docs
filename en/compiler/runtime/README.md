# Runtime Environment and Standard Library Services (`src/runtime/`)

A collection of documents defining the system logic classes mapped to native services for built-in abstract base collections, or exposed operational modules functioning via a system interface API acting within the runtime instances.

## Directory Index

- **[runtime_core.md](runtime_core.md):** The entry point pipelines for converting isolated parameter modules into assigned outputs wrapper via API. Acts as the functional registry for the FFI referencing tables mapped across the standard operational environment.
- **[runtime_services.md](runtime_services.md):** Integrated link controllers handling time security layers, networking web protocols, cryptography mapping for asymmetric arrays and symmetric configurations, as well IO output routing control modules validating standard output procedures.
- **[runtime_collections.md](runtime_collections.md):** Assistant algorithm instances supporting native array operations, reference maps for keys, unqiue blocks for sets, automated JSON parameter conversion testing, and relational base transactions via cyclical modular execution iterations (Fibers).
