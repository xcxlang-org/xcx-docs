# XCX 3.1 — операторы

## Арифметические операторы

| Оператор | Описание                         | Пример     | Результат |
|----------|----------------------------------|------------|-----------|
| `+`      | Сложение                         | `10 + 3`   | `13`      |
| `-`      | Вычитание / унарный минус        | `10 - 3`   | `7`       |
| `*`      | Умножение                        | `10 * 3`   | `30`      |
| `/`      | Деление (для `i` — усечение)     | `10 / 3`   | `3`       |
| `%`      | Остаток от деления               | `10 % 3`   | `1`       |
| `^`      | Возведение в степень             | `2 ^ 10`   | `1024`    |
| `++`     | Конкатенация цифр целого числа   | `48 ++ 77` | `4877`    |

## Строковые операторы

| Оператор | Описание              | Пример                   | Результат        |
|----------|-----------------------|--------------------------|------------------|
| `+`      | Конкатенация          | `"Hi " + "User"`         | `"Hi User"`      |
| `HAS`    | Поиск подстроки       | `"Search" HAS "ear"`     | `true`           |

> [!TIP]
> При конкатенации строк числа и логические значения автоматически преобразуются в строки (например, `"Value: " + 42` → `"Value: 42"`).

## Логические операторы

| Оператор | Синоним | Описание      |
|----------|---------|---------------|
| `AND`    | `&&`    | Логическое И  |
| `OR`     | `||`    | Логическое ИЛИ|
| `NOT`    | `!!`    | Логическое НЕ |

## Операторы сравнения

Используются для сравнения значений одного типа.

| Оператор | Описание           |
|----------|--------------------|
| `==`     | Равно              |
| `!=`     | Не равно           |
| `<`      | Меньше             |
| `>`      | Больше             |
| `<=`     | Меньше или равно   |
| `>=`     | Больше или равно   |

## Операторы над множествами

| Слово                  | Символ | Описание                    |
|------------------------|--------|-----------------------------|
| `UNION`                | `∪`    | Объединение множеств        |
| `INTERSECTION`         | `∩`    | Пересечение                 |
| `DIFFERENCE`           | `\`    | Разность                    |
| `SYMMETRIC_DIFFERENCE` | `⊕`    | Симметрическая разность     |

## Приоритет операторов (от высшего к низшему)

1. Вызовы функций, методов, индексация `[]`
2. Унарный минус `-`, `!!` (NOT)
3. Возведение в степень `^`
4. Умножение `*`, деление `/`, остаток `%`
5. Операции над множествами `UNION`, `INTERSECTION`, `DIFFERENCE`, `SYM_DIFF`
6. Сложение `+`, вычитание `-`, конкатенация `++`
7. Сравнения `HAS`, `>`, `<`, `>=`, `<=`
8. Равенство `==`, `!=`
9. Логика `AND`, `&&`
10. Логика `OR`, `||`
