# XCX 3.1 — переменные и константы

## Объявление переменных

Переменные объявляются синтаксисом `type: name = value;`.

```xcx
i: age = 25;
f: pi = 3.14159;
s: name = "XCX";
b: is_ready = true;
```

Переменную можно объявить без начального значения — тогда используется значение по умолчанию для типа (например, `0` для `i`, `""` для `s`).

## Переназначение

После объявления переменной можно присвоить новое значение оператором `=`.

```xcx
i: count = 0;
count = count + 1;
```

## Константы

Константы объявляются ключевым словом `const`. Их нужно инициализировать при объявлении; после этого изменить нельзя.

```xcx
const i: MAX_CONNECTIONS = 1024;
const s: VERSION = "2.0.0";
```

## Затенение переменных запрещено

XCX **не** поддерживает затенение переменных. Объявление переменной с тем же именем в любой области видимости — включая вложенный блок — это **ошибка компиляции** (`RedefinedVariable`).

```xcx
i: x = 10;
if (true) then;
    i: x = 20;   --- COMPILE ERROR: RedefinedVariable
end;
```

Чтобы изменить значение внутри блока, используйте переназначение, а не повторное объявление:

```xcx
i: x = 10;
if (true) then;
    x = 20;      --- OK: reassignment to existing variable
    >! x;        --- 20
end;
>! x;            --- 20 (the global variable was modified)
```
