# XCX 3.1 — основы синтаксиса

## Комментарии

```xcx
--- Single-line comment
i: price = 100;  --- inline comment

---
Multi-line
comment block.
*---
```

## Идентификаторы

Идентификаторы **чувствительны к регистру** и должны соответствовать шаблону `[a-zA-Z][a-zA-Z0-9_]*`.

| Допустимо     | Недопустимо | Причина                              |
|---------------|-------------|--------------------------------------|
| `userData`    | `1stUser`   | Не может начинаться с цифры          |
| `user_data`   | `user-data` | Дефис — оператор минус               |
| `counter1`    | `user name` | Пробелы не допускаются               |

Ключевые слова (`if`, `func`, `fiber`, `i`, `f`, `s`, `b` и т.д.) зарезервированы.

## Завершение инструкций

Инструкции завершаются точкой с запятой `;`. Это относится к объявлениям переменных, вызовам функций, присваиваниям и директивам.

```xcx
i: x = 10;
>! "Hello";
```

## Блоки

Блоки открываются ключевым словом (например, `then`, `do`, `{`) и закрываются `end;`.

```xcx
if (x > 0) then;
    >! "Positive";
end;
```
