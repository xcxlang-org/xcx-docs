# XCX 3.1 — методы строк

Строки в XCX неизменяемы. Методы возвращают **новую строку** и не изменяют исходную.

## Свойства

- `.length`: число кодовых точек Unicode в строке (например, `"zażółć".length` равно 6). **Вызывается без скобок.**

## Escape-последовательности

В строковых литералах поддерживаются стандартные escape-последовательности:

| Последовательность | Эффект              |
|--------------------|---------------------|
| `\n`               | Перевод строки      |
| `\t`               | Горизонтальная табуляция |
| `\r`               | Возврат каретки     |
| `\"`               | Двойная кавычка     |
| `\\`               | Обратная косая черта |
| `\xNN`             | Символ в hex (например, `\x1b`) |
| `\NNN`             | Символ в восьмеричной (например, `\033`) |

## Методы

| Метод                | Сигнатура    | Описание                                           |
|----------------------|--------------|----------------------------------------------------|
| `.upper()`           | `() → s`     | Все символы в верхний регистр.                     |
| `.lower()`           | `() → s`     | Все символы в нижний регистр.                     |
| `.trim()`            | `() → s`     | Удаляет пробелы в начале и конце.                  |
| `.replace(f, t)`     | `(s, s) → s` | Заменяет все вхождения `f` на `t`.                 |
| `.slice(s, e)`       | `(i, i) → s` | Подстрока с индекса `s` до `e`.                    |
| `.indexOf(s)`        | `(s) → i`    | Индекс первого вхождения или -1.                   |
| `.lastIndexOf(s)`    | `(s) → i`    | Индекс последнего вхождения или -1.                |
| `.startsWith(s)`     | `(s) → b`    | `true`, если строка начинается с `s`.              |
| `.endsWith(s)`       | `(s) → b`    | `true`, если строка заканчивается на `s`.          |
| `.toInt()`           | `() → i`     | Разбор в целое; при ошибке — `halt.error`.         |
| `.toFloat()`         | `() → f`     | Разбор в вещественное; при ошибке — `halt.error`.  |
| `.split(s)`          | `(s) → array:s` | Разделение по разделителю `s`; массив строк.   |

## Примеры

```xcx
s: raw = "  XCX-Language  ";
s: clean = raw.trim().lower().replace("-", "_"); --- "xcx_language"

i: start = "Programming".indexOf("gram");       --- 3
s: part = "Programming".slice(0, 4);            --- "Prog"

i: age = "25".toInt();
array:s: parts = "a,b,c".split(",");           --- {"a", "b", "c"}
```
