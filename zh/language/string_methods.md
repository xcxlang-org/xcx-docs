# XCX 3.1 字符串方法

XCX 中的字符串对象是不可变的。方法返回**新字符串**，不会修改原字符串。

## 属性

- `.length`：返回字符串中 Unicode 码点的数量（例如 `"zażółć".length` 为 6）。**使用时无需括号。**

## 转义序列

字符串字面量支持标准转义序列：

| 序列   | 效果           |
|--------|----------------|
| `\n`   | 换行           |
| `\t`   | 水平制表符     |
| `\r`   | 回车           |
| `\"`   | 双引号         |
| `\\`   | 反斜杠         |
| `\xNN` | 十六进制字符（如 `\x1b`） |
| `\NNN` | 八进制字符（如 `\033`）   |

## 方法

| 方法                 | 签名         | 说明                                   |
|----------------------|--------------|----------------------------------------|
| `.upper()`           | `() → s`     | 将所有字符转换为大写。                 |
| `.lower()`           | `() → s`     | 将所有字符转换为小写。                 |
| `.trim()`            | `() → s`     | 去除首尾空白字符。                     |
| `.replace(f, t)`     | `(s, s) → s` | 将所有 `f` 替换为 `t`。                |
| `.slice(s, e)`       | `(i, i) → s` | 返回从索引 `s` 到 `e` 的子串。         |
| `.indexOf(s)`        | `(s) → i`    | 返回首次出现的索引，未找到则为 -1。    |
| `.lastIndexOf(s)`    | `(s) → i`    | 返回末次出现的索引，未找到则为 -1。    |
| `.startsWith(s)`     | `(s) → b`    | 若字符串以 `s` 开头则返回 true。       |
| `.endsWith(s)`       | `(s) → b`    | 若字符串以 `s` 结尾则返回 true。       |
| `.toInt()`           | `() → i`     | 解析为整数；失败时 `halt.error`。      |
| `.toFloat()`         | `() → f`     | 解析为浮点数；失败时 `halt.error`。    |
| `.split(s)`          | `(s) → array:s` | 按分隔符 `s` 拆分；返回字符串数组。 |

## 示例

```xcx
s: raw = "  XCX-Language  ";
s: clean = raw.trim().lower().replace("-", "_"); --- "xcx_language"

i: start = "Programming".indexOf("gram");       --- 3
s: part = "Programming".slice(0, 4);            --- "Prog"

i: age = "25".toInt();
array:s: parts = "a,b,c".split(",");           --- {"a", "b", "c"}
```
