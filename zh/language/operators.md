# XCX 3.1 运算符

## 算术运算符

| 运算符 | 说明                     | 示例       | 结果    |
|--------|--------------------------|------------|---------|
| `+`    | 加法                     | `10 + 3`   | `13`    |
| `-`    | 减法 / 一元取负          | `10 - 3`   | `7`     |
| `*`    | 乘法                     | `10 * 3`   | `30`    |
| `/`    | 除法（`i` 类型截断）     | `10 / 3`   | `3`     |
| `%`    | 取模（余数）             | `10 % 3`   | `1`     |
| `^`    | 幂运算                   | `2 ^ 10`   | `1024`  |
| `++`   | 整数数字拼接             | `48 ++ 77` | `4877`  |

## 字符串运算符

| 运算符 | 说明         | 示例                     | 结果             |
|--------|--------------|--------------------------|------------------|
| `+`    | 字符串连接   | `"Hi " + "User"`         | `"Hi User"`      |
| `HAS`  | 子串搜索     | `"Search" HAS "ear"`     | `true`           |

> [!TIP]
> 字符串连接会自动将数字和布尔值转换为字符串（例如 `"Value: " + 42` → `"Value: 42"`）。

## 逻辑运算符

| 运算符 | 别名  | 说明     |
|--------|-------|----------|
| `AND`  | `&&`  | 逻辑与   |
| `OR`   | `||`  | 逻辑或   |
| `NOT`  | `!!`  | 逻辑非   |

## 比较运算符

用于比较相同类型的值。

| 运算符 | 说明       |
|--------|------------|
| `==`   | 等于       |
| `!=`   | 不等于     |
| `<`    | 小于       |
| `>`    | 大于       |
| `<=`   | 小于等于   |
| `>=`   | 大于等于   |

## 集合运算符

| 关键字                 | 符号 | 说明       |
|------------------------|------|------------|
| `UNION`                | `∪`  | 集合并     |
| `INTERSECTION`         | `∩`  | 集合交     |
| `DIFFERENCE`           | `\`  | 集合差     |
| `SYMMETRIC_DIFFERENCE` | `⊕`  | 对称差     |

## 运算符优先级（从高到低）

1. 函数调用、方法调用、索引 `[]`
2. 一元取负 `-`、`!!`（NOT）
3. 幂运算 `^`
4. 乘法 `*`、除法 `/`、取模 `%`
5. 集合运算 `UNION`、`INTERSECTION`、`DIFFERENCE`、`SYM_DIFF`
6. 加法 `+`、减法 `-`、拼接 `++`
7. 比较 `HAS`、`>`、`<`、`>=`、`<=`
8. 相等性 `==`、`!=`
9. 逻辑 `AND`、`&&`
10. 逻辑 `OR`、`||`
