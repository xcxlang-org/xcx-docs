# XCX 3.1 構文の基本

## コメント

```xcx
--- 1行コメント
i: price = 100;  --- インラインコメント

---
マルチラインコメント
ブロック
*---
```

## 識別子

識別子は **大文字と小文字を区別** し、パターン `[a-zA-Z][a-zA-Z0-9_]*` に一致する必要があります。

| 有効       | 無効         | 理由                       |
|-----------|-------------|------------------------------|
| `userData`    | `1stUser`    | 数字で始まることはできません    |
| `user_data`   | `user-data`  | ハイフンはマイナス演算子です    |
| `counter1`    | `user name`  | スペースは許可されていません       |

キーワード（`if`、`func`、`fiber`、`i`、`f`、`s`、`b` など）は予約済みです。

## ステートメント終了記号

ステートメントはセミコロン `;` で終了します。これは変数宣言、関数呼び出し、割り当て、およびディレクティブに適用されます。

```xcx
i: x = 10;
>! "Hello";
```

## ブロック

ブロックはキーワード（例：`then`、`do`、`{`）で開かれ、`end;` で閉じられます。

```xcx
if (x > 0) then;
    >! "Positive";
end;
```
