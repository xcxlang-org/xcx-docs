# XCX 3.1 演算子

## 算術演算子

| 演算子 | 説明                         | 例         | 結果    |
|--------|------------------------------|------------|---------|
| `+`    | 加算                         | `10 + 3`   | `13`    |
| `-`    | 減算 / 単項否定              | `10 - 3`   | `7`     |
| `*`    | 乗算                         | `10 * 3`   | `30`    |
| `/`    | 除算（`i` では切り捨て）     | `10 / 3`   | `3`     |
| `%`    | 剰余（モジュロ）             | `10 % 3`   | `1`     |
| `^`    | 累乗                         | `2 ^ 10`   | `1024`  |
| `++`   | 整数の桁連結                 | `48 ++ 77` | `4877`  |

## 文字列演算子

| 演算子 | 説明               | 例                       | 結果             |
|--------|--------------------|--------------------------|------------------|
| `+`    | 連結               | `"Hi " + "User"`         | `"Hi User"`      |
| `HAS`  | 部分文字列の検索   | `"Search" HAS "ear"`     | `true`           |

> [!TIP]
> 文字列連結時、数値と真偽値は自動的に文字列に変換されます（例：`"Value: " + 42` → `"Value: 42"`）。

## 論理演算子

| 演算子 | 別名  | 説明       |
|--------|-------|------------|
| `AND`  | `&&`  | 論理 AND   |
| `OR`   | `||`  | 論理 OR    |
| `NOT`  | `!!`  | 論理 NOT   |

## 比較演算子

同じ型の値を比較する際に使用します。

| 演算子 | 説明           |
|--------|----------------|
| `==`   | 等しい         |
| `!=`   | 等しくない     |
| `<`    | より小さい     |
| `>`    | より大きい     |
| `<=`   | 以下           |
| `>=`   | 以上           |

## 集合演算子

| キーワード             | 記号 | 説明           |
|------------------------|------|----------------|
| `UNION`                | `∪`  | 和集合         |
| `INTERSECTION`         | `∩`  | 積集合         |
| `DIFFERENCE`           | `\`  | 差集合         |
| `SYMMETRIC_DIFFERENCE` | `⊕`  | 対称差         |

## 演算子の優先順位（高い順）

1. 関数呼び出し、メソッド呼び出し、インデックス `[]`
2. 単項否定 `-`、`!!`（NOT）
3. 累乗 `^`
4. 乗算 `*`、除算 `/`、剰余 `%`
5. 集合演算 `UNION`、`INTERSECTION`、`DIFFERENCE`、`SYM_DIFF`
6. 加算 `+`、減算 `-`、連結 `++`
7. 比較 `HAS`、`>`、`<`、`>=`、`<=`
8. 等値 `==`、`!=`
9. 論理 `AND`、`&&`
10. 論理 `OR`、`||`
