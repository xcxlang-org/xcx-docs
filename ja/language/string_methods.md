# XCX 3.1 文字列メソッド

XCX の文字列オブジェクトは不変です。メソッドは **新しい文字列** を返し、元の文字列は変更しません。

## プロパティ

- `.length`: 文字列内の Unicode コードポイント数を返します（例: `"zażółć".length` は 6）。**括弧は付けません。**

## エスケープシーケンス

文字列リテラルは標準的なエスケープシーケンスをサポートします。

| シーケンス | 効果 |
|----------|--------|
| `\n`     | 改行 |
| `\t`     | 水平タブ |
| `\r`     | キャリッジリターン |
| `\"`     | 二重引用符 |
| `\\`     | バックスラッシュ |
| `\xNN`   | 16 進文字（例: `\x1b`） |
| `\NNN`   | 8 進文字（例: `\033`） |

## メソッド

| メソッド               | シグネチャ    | 説明                                      |
|----------------------|--------------|--------------------------------------------------|
| `.upper()`           | `() → s`     | すべての文字を大文字に変換。            |
| `.lower()`           | `() → s`     | すべての文字を小文字に変換。            |
| `.trim()`            | `() → s`     | 先頭・末尾の空白を削除。             |
| `.replace(f, t)`     | `(s, s) → s` | `f` の出現をすべて `t` に置換。        |
| `.slice(s, e)`       | `(i, i) → s` | インデックス `s` から `e` までの部分文字列を返す。      |
| `.indexOf(s)`        | `(s) → i`    | 最初の出現位置のインデックス。なければ -1。        |
| `.lastIndexOf(s)`    | `(s) → i`    | 最後の出現位置のインデックス。なければ -1。         |
| `.startsWith(s)`     | `(s) → b`    | 文字列が `s` で始まるとき true。          |
| `.endsWith(s)`       | `(s) → b`    | 文字列が `s` で終わるとき true。            |
| `.toInt()`           | `() → i`     | 文字列を整数にパース。失敗時は `halt.error`。  |
| `.toFloat()`         | `() → f`     | 文字列を浮動小数にパース。失敗時は `halt.error`。    |
| `.split(s)`          | `(s) → array:s` | 区切り文字 `s` で分割。文字列の配列を返す。 |

## 例

```xcx
s: raw = "  XCX-Language  ";
s: clean = raw.trim().lower().replace("-", "_"); --- "xcx_language"

i: start = "Programming".indexOf("gram");       --- 3
s: part = "Programming".slice(0, 4);            --- "Prog"

i: age = "25".toInt();
array:s: parts = "a,b,c".split(",");           --- {"a", "b", "c"}
```
